Cloud DLP System

A Data Loss Prevention system for scanning uploads, encrypting allowed content, and maintaining secure audit logs with integrity verification.

Project modules

  • Module 1: Rule-Based Content Inspection — regex-based detection of PII (email, phone, password patterns).
  • Module 2: Cryptographic Hashing for Integrity — SHA-256 of encrypted content for tamper detection.
  • Module 3: Secure Violation Logging — audit trail for every upload attempt.
  • Module 4: User-Defined Security Policies — configurable detection rules (see Policies).
  • Module 5: Performance & Cloud Integration — encrypted storage, optional cloud backend.

Tech stack

Backend: FastAPI, SQLAlchemy, Cryptography (Fernet), SQLite. Frontend: vanilla HTML/CSS/JS. DLP: regex policies (JSON).
