# Documentation

Organized reference and project docs for Cloud DLP.

## Structure

| Folder | Contents |
|--------|----------|
| **guides/** | How-to and design docs |
| **paper/** | LaTeX paper source and PDF |
| **presentations/** | Slide decks and notes |
| **thesis/** | Thesis / project report |
| **journal/** | Journal submission |

## Guides

- **[ARCHITECTURE.md](guides/ARCHITECTURE.md)** — System design, data flow, components, and extending the app (also at project root as `ARCHITECTURE.md`).
- **[END_TO_END_FLOW.md](guides/END_TO_END_FLOW.md)** — End-to-end flow from UI through backend to storage: upload, DLP scan, crypto, assets, logs.
- **[DEMO.md](guides/DEMO.md)** — How to run the demo (backend, demo script, browser).

## Paper

- **paper/** — LaTeX source (`cloud_dlp_paper.tex`) and build artifacts (`.aux`, `.log`, `.out`) plus compiled **cloud_dlp_paper.pdf**.

## Presentations

- **presentations/** — Slide decks (`.pptx`, `.pdf`) and **ppt.md** (short bullet summary of slide content).

## Other

- **thesis/project_report.docx** — Thesis or project report document.
- **journal/paper.docx** — Journal submission document.
